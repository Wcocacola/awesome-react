# Awesome React

Webpack 2 automatically splits routes and loads them on demand.

## Contains

- [x] [Webpack 2.2](https://webpack.github.io)
- [x] [React 0.15](https://facebook.github.io/react/)
- [x] [Babel 6.x](https://babeljs.io/)

## Setup

```
npm run dev
```

## Running

```
$ npm start
```

## Build

```
$ npm run build
```
